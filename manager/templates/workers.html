How to Run a Worker
  1. Make sure Python 3.8+ is installed
  2. Install dependencies: pip install psutil docker
  3. (Optional) Install Docker for sandboxed execution
  4. Navigate to the worker directory
  5. Run: python client.py <manager_ip> <your_username>

Your machine will automatically detect its specs and start processing jobs. You'll earn credits for each job completed!
